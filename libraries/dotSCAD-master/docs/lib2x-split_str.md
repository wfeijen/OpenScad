# split_str

The dir changed since 2.0. 

Splits the given string around matches of the given delimiting character. 

## Parameters

- `t` : The source string.
- `delimiter` : The delimiting character.

## Examples
    
	use <util/split_str.scad>;
	
	echo(split_str("hello,world", ","));  // ECHO: ["hello", "world"]
